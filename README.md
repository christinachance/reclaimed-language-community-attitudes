# reclaimed-language-community-attitudes


This is the repo for the figure and table generation for the paper **"IYKYK (But AI Doesn’t): Automated Content Moderation Does Not Capture
Communities’ Heterogeneous Attitudes Towards Reclaimed Language"**. The data is not made public due to the content. If you are interested in the data, please reach out to Christina Chance at christinachance315@gmail.com sharing the goal and potential usage of the data.
